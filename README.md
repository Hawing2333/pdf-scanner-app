# PDF 图片文字搜索 (Android)

识别PDF中图片形式的文字，输入关键词搜索并定位到对应页码。

## 功能

- **PDF 文字识别**：使用 Google ML Kit OCR 引擎，支持中英文混合识别
- **关键词搜索**：输入关键词，快速定位所有匹配位置及页码
- **正则表达式**：支持正则表达式搜索和区分大小写
- **PDF 预览**：点击搜索结果可跳转到 PDF 对应页面
- **PDF 浏览**：支持上下翻页浏览 PDF 全部内容

## 技术方案

| 组件 | 技术 | 说明 |
|------|------|------|
| OCR 引擎 | Google ML Kit (Chinese) | 谷歌官方中文文字识别，轻量高效 |
| PDF 渲染 | Android PdfRenderer | 系统原生渲染，无需额外依赖 |
| UI 框架 | Java + Material Design 3 | 原生 Android 界面，流畅美观 |

## 本地部署

### 环境要求

- Android Studio Ladybug (2024.2) 或更高版本
- JDK 21（通过 Gradle 自动下载）
- Android SDK 36 (API 36)
- 最低支持 Android 11 (API 30)

### 构建步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/Hawing2333/pdf-scanner-app.git
   ```

2. **用 Android Studio 打开**
   - 选择 `File` → `Open`
   - 选择 `pdf-scanner-app` 目录
   - 等待 Gradle 同步完成

3. **构建 APK**
   - 点击菜单 `Build` → `Build Bundle(s) / APK(s)` → `Build APK(s)`
   - 或使用命令行：
     ```bash
     cd pdf-scanner-app
     ./gradlew assembleDebug
     ```

4. **安装到设备**
   - 连接 Android 设备或启动模拟器
   - 点击 `Run` (绿色三角按钮)
   - APK 位置：`app/build/outputs/apk/debug/app-debug.apk`

## 使用方法

### 第一步：选择 PDF 文件

点击 **「选择PDF」** 按钮，从手机中选择需要搜索的 PDF 文件。

### 第二步：开始识别

选择文件后，点击 **「开始识别」** 按钮。软件会逐页识别 PDF 中的文字，进度条显示当前识别进度（如：`正在识别第 2/5 页...`）。

识别完成后提示：`识别完成！共 5 页`

### 第三步：搜索关键词

在搜索框中输入要搜索的关键词，点击 **「搜索」** 按钮（或按键盘回车）。

搜索结果会显示在下方列表，包含：
- **页码**：蓝色圆形标记显示匹配所在页
- **上下文**：匹配位置的前后文字，关键词用 `[ ]` 高亮

可选设置：
- **区分大小写**：勾选后搜索区分大小写
- **正则表达式**：勾选后支持正则表达式搜索

### 第四步：跳转预览

点击搜索结果中的任意一条，会打开 PDF 预览页面并自动跳转到对应页码。也可以点击 **「查看 PDF 预览」** 按钮浏览 PDF 全部内容。

PDF 预览中支持：
- **上一页 / 下一页** 按钮翻页
- **滑动浏览** 全部页面

## 注意事项

- 首次使用 ML Kit 时，Google Play 服务会自动下载 OCR 模型（约 30MB），需联网
- 识别速度取决于 PDF 页数和图片复杂度，建议先用少量页面测试
- 应用需要「读取存储文件」权限，首次使用时会请求授权
- 支持 Android 11 (API 30) 及以上版本
